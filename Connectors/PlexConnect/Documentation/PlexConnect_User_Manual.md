# PlexConnect - User Manual

## What is PlexConnect?

PlexConnect is a ShareConnect application that connects your Android device to Plex Media Servers. It provides seamless access to your personal media library, allowing you to browse movies, TV shows, music, and photos from anywhere.

## Getting Started

### Installation

1. Install PlexConnect APK on your Android device
2. Launch the app
3. Configure your Plex server connection

### First-Time Setup

1. **Server Configuration**:
   - Open Settings tab
   - Enter your Plex server URL (e.g., http://192.168.1.100:32400)
   - Sign in with your Plex account
   - Test the connection

2. **Security (Optional)**:
   - Enable PIN or biometric authentication
   - Configure in Settings > Security

## Features

### Libraries Tab

Browse and manage your Plex libraries:

- **View Libraries**: See all accessible media libraries (Movies, TV Shows, Music, Photos)
- **Library Info**: Tap any library to view details and content count
- **Recently Added**: Quick access to newest content
- **Storage Info**: View library sizes and server information

### Media Browser

Navigate your media collection:

- **Movies**: Browse by title, year, genre, or rating
- **TV Shows**: Explore series, seasons, and episodes
- **Music**: Access artists, albums, and playlists
- **Photos**: View photo libraries and albums

### Media Details

View detailed information about any item:

- **Metadata**: Title, summary, cast, crew, ratings
- **Artwork**: High-resolution posters and backgrounds
- **Playback Info**: Duration, resolution, audio format
- **Watch Status**: Track what you've watched

### Search Functionality

Find content quickly:

- **Global Search**: Search across all libraries
- **Library-Specific**: Search within specific libraries
- **Filters**: Filter by type, year, genre, or rating
- **Recent Searches**: Quick access to previous searches

### Playback Features

Manage your viewing experience:

- **Watch Status**: Mark items as watched/unwatched
- **Progress Tracking**: Resume where you left off
- **Ratings**: Rate movies and shows
- **Collections**: Browse curated collections

## Troubleshooting

### Connection Issues

**Problem**: Cannot connect to Plex server
**Solutions**:
- Verify server URL and port (default: 32400)
- Check network connectivity
- Ensure Plex server is running
- Verify firewall settings

**Problem**: Authentication fails
**Solutions**:
- Check Plex account credentials
- Verify server allows external connections
- Try manual token generation
- Check server authentication settings

### Performance Issues

**Problem**: Slow loading or timeouts
**Solutions**:
- Check server hardware resources
- Optimize network connection
- Reduce library scan frequency
- Clear app cache and data

**Problem**: High data usage
**Solutions**:
- Enable data saver mode
- Reduce artwork quality
- Limit background sync
- Use Wi-Fi when possible

### Content Issues

**Problem**: Missing libraries or content
**Solutions**:
- Refresh libraries in Plex server
- Check library permissions
- Verify content is properly scanned
- Rebuild server database if necessary

**Problem**: Incorrect metadata
**Solutions**:
- Refresh metadata in Plex server
- Check agent settings
- Verify file naming conventions
- Manual metadata correction

## FAQ

### Q: Do I need a Plex account to use PlexConnect?
**A**: Yes, you need a Plex account and access to a Plex Media Server.

### Q: Can I use PlexConnect with multiple servers?
**A**: Yes, PlexConnect supports multiple Plex servers simultaneously.

### Q: Does PlexConnect support encrypted connections?
**A**: Yes, PlexConnect supports HTTPS connections for secure data transmission.

### Q: Can I control playback through PlexConnect?
**A**: PlexConnect focuses on browsing and managing your library. For playback, it integrates with other ShareConnect apps.

### Q: How often does PlexConnect sync with the server?
**A**: Sync frequency can be configured in settings, with options for real-time, hourly, or manual sync.

### Q: Is my viewing history shared with other ShareConnect apps?
**A**: Yes, if you enable history sync, your viewing activity can be shared across the ShareConnect ecosystem.

### Q: What file formats does PlexConnect support?
**A**: PlexConnect supports all formats that your Plex server can handle, as it works through the Plex API.

### Q: Can I use PlexConnect without an internet connection?
**A**: Some cached information is available offline, but full functionality requires server connectivity.
