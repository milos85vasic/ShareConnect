Getting Started with ShareConnect

Your complete guide to installing and configuring ShareConnect

1

Download & Install

Download ShareConnect from our GitHub releases page. Choose the appropriate APK for your Android version.

# Download the latest release
wget https://github.com/yourusername/ShareConnect/releases/latest/download/ShareConnector-debug.apk

# Install on your Android device
adb install ShareConnector-debug.apk
Note: Enable "Install from unknown sources" in your Android settings if installing manually.
2

First Launch & Permissions

When you first launch ShareConnect, you'll need to grant necessary permissions:

  • Internet Access: Required for connecting to your services
  • Storage Access: For caching and temporary files
  • Notification Access: For download progress updates

The app will guide you through the permission setup process.

3

Configure Your First Connector

ShareConnect works by connecting to your existing services. Let's set up your first connector:

  1. Open ShareConnect and tap "Add Connector"
  2. Choose your service type (Torrent, Media Server, etc.)
  3. Enter your service details (IP address, port, credentials)
  4. Test the connection
  5. Save and you're ready to share!
4

Share Your First Media Link

Now that you have a connector configured, let's share some media:

  1. Find a media link in your browser or app
  2. Tap the share button
  3. Select "ShareConnect" from the share menu
  4. Choose your configured connector
  5. Watch as ShareConnect handles the rest!
Pro Tip: ShareConnect automatically detects the content type and suggests the best connector.

qBittorrent Connector Setup

Professional torrent client integration with advanced features like RSS feeds, search plugins, and automated downloading.

Difficulty: Easy Setup Time: 5 minutes

Prerequisites

  • qBittorrent Installation: qBittorrent must be installed and running on your network
  • Web UI Enabled: Web interface must be enabled in qBittorrent preferences
  • Network Access: Your Android device must be able to reach the qBittorrent host
  • Credentials: Web UI username and password (if authentication is enabled)

qBittorrent Configuration

  1. Open qBittorrent on your computer
  2. Go to Tools → Preferences → Web UI
  3. Check "Enable the Web User Interface (Remote control)"
  4. Set a port (default 8080) and optional username/password
  5. Click Apply and restart qBittorrent if prompted

ShareConnect Setup

  1. Open ShareConnect on your Android device
  2. Tap "Add Connector"
  3. Select "qBittorrent" from the list
  4. Enter connection details:
    • Host: IP address or hostname of your qBittorrent machine
    • Port: Web UI port (usually 8080)
    • Username: Web UI username (leave blank if not set)
    • Password: Web UI password (leave blank if not set)
  5. Tap "Test Connection"
  6. If successful, tap "Save"

Testing the Connection

To test your qBittorrent connector:

  1. Find a torrent file or magnet link
  2. Use Android's share menu and select ShareConnect
  3. Choose your qBittorrent connector
  4. The torrent should appear in qBittorrent and start downloading

Troubleshooting

Connection Failed
  • Verify qBittorrent is running
  • Check IP address and port
  • Ensure Web UI is enabled
  • Try disabling Windows Firewall temporarily
Authentication Error
  • Verify username and password
  • Check if authentication is required in qBittorrent settings
  • Try creating a new user account in qBittorrent
Torrents Not Starting
  • Check available disk space
  • Verify download directory permissions
  • Check qBittorrent logs for errors

Advanced Features

  • RSS Feeds: Automatic downloading from RSS feeds
  • Search Plugins: Search for torrents directly from qBittorrent
  • Categories: Organize downloads by category
  • Bandwidth Limits: Control download/upload speeds

Transmission Connector Setup

Lightweight and efficient torrent client perfect for headless servers and resource-constrained environments.

Difficulty: Medium Setup Time: 10 minutes

Prerequisites

  • Transmission Installation: Transmission daemon (transmission-daemon) running
  • RPC Access: RPC interface enabled and accessible
  • Configuration: Access to Transmission settings.json file
  • Network: Android device can reach Transmission RPC port (default 9091)

Transmission Configuration

For Linux/macOS:

# Stop transmission-daemon
sudo systemctl stop transmission-daemon

# Edit settings.json
sudo nano /var/lib/transmission-daemon/info/settings.json

# Find and set:
"rpc-enabled": true,
"rpc-port": 9091,
"rpc-whitelist-enabled": false,
"rpc-authentication-required": true,
"rpc-username": "your-username",
"rpc-password": "your-password"

# Start transmission-daemon
sudo systemctl start transmission-daemon

ShareConnect Setup

  1. Open ShareConnect
  2. Tap "Add Connector"
  3. Select "Transmission"
  4. Enter connection details:
    • Host: Server IP address
    • Port: RPC port (default 9091)
    • Username: RPC username
    • Password: RPC password
  5. Test connection and save

Transmission Advantages

  • Lightweight: Minimal resource usage
  • Headless: Perfect for servers
  • Watch Directory: Automatic torrent loading
  • Scriptable: Easy automation

Plex Media Server Setup

Complete integration with Plex Media Server for streaming your media collection anywhere.

Difficulty: Easy Setup Time: 5 minutes

Prerequisites

  • Plex Server: Plex Media Server running and accessible
  • Plex Account: Valid Plex account with server access
  • Network Access: Android device can reach Plex server
  • Libraries: Media libraries configured in Plex

ShareConnect Setup

  1. Open ShareConnect
  2. Tap "Add Connector"
  3. Select "Plex Media Server"
  4. Enter server details:
    • Server Address: IP address and port (e.g., 192.168.1.100:32400)
    • Username: Your Plex account email
    • Password: Your Plex account password
  5. Select target library for uploads
  6. Test connection and save

Using Plex Connector

The Plex connector allows you to:

  • Direct Upload: Send media files directly to Plex libraries
  • Auto-Organization: Files are automatically categorized and metadata fetched
  • Cross-Device Sync: Access your media on any Plex-enabled device
  • Transcoding: Automatic format conversion for compatibility

JDownloader Setup

Premium download manager integration supporting 1800+ sites with MyJDownloader service.

Difficulty: Easy Setup Time: 5 minutes

Prerequisites

  • JDownloader: Installed and running on your computer
  • MyJDownloader: Account created at my.jdownloader.org
  • Premium Accounts: Optional premium account credentials
  • Network: Android device can reach MyJDownloader service

ShareConnect Setup

  1. Open ShareConnect
  2. Tap "Add Connector"
  3. Select "JDownloader"
  4. Enter MyJDownloader credentials:
    • Email: Your MyJDownloader account email
    • Password: Your MyJDownloader password
  5. Test connection and save

JDownloader Features

  • 1800+ Sites: Support for virtually every download site
  • Premium Support: Automatic premium account usage
  • CAPTCHA Solving: Automatic captcha resolution
  • Batch Downloads: Handle multiple links simultaneously
  • Link Checking: Verify download availability

Troubleshooting Guide

Common issues and their solutions

Connection Failed

Symptoms: Unable to connect to your service

Solutions:

  • Verify IP address and port are correct
  • Check network connectivity
  • Ensure service is running and accessible
  • Disable firewall temporarily for testing

Authentication Error

Symptoms: Login credentials rejected

Solutions:

  • Double-check username and password
  • Ensure account has necessary permissions
  • Try resetting password if applicable
  • Check if two-factor authentication is enabled

Share Not Working

Symptoms: Share button doesn't show ShareConnect

Solutions:

  • Ensure ShareConnect is installed and not disabled
  • Clear app cache and restart
  • Check Android share settings
  • Reinstall ShareConnect if necessary

Performance Issues

Symptoms: Slow response or high battery usage

Solutions:

  • Check network connection speed
  • Close unused connectors
  • Clear app cache
  • Update to latest version

Frequently Asked Questions

Quick answers to common questions

Is ShareConnect free?

Yes! ShareConnect is completely free and open source. All connectors are included at no cost.

Does ShareConnect store my data?

No. ShareConnect acts as a bridge between your apps and services. Your media and personal data never passes through our servers.

Which Android versions are supported?

ShareConnect supports Android API level 21 and above (Android 5.0+), with optimized performance on Android 8.0+.

Can I use multiple connectors simultaneously?

Absolutely! You can configure multiple connectors of the same type and ShareConnect will let you choose which one to use for each share.

Is my data secure?

ShareConnect uses industry-standard encryption for all communications. We recommend using HTTPS wherever possible for maximum security.

What if my service isn't supported?

ShareConnect is constantly growing! Check our GitHub issues for requested connectors or submit a feature request.
