Powerful Features

Everything you need to manage your self-hosted services

🎨

Unified Design

Material Design 3 with 6 built-in themes plus custom theme creator. Dark mode support across all apps.

🔄

Real-Time Sync

Asinka-powered synchronization. Profiles, themes, history shared instantly across all 20 applications.

🔐

Security First

PIN authentication, biometric support, encrypted sync. No credentials stored locally. HTTPS everywhere.

⚡

Performance

Cold start <2s, memory <150MB, battery <5%/hr. Optimized for efficiency on all devices.

📱

Native Android

Built with Jetpack Compose and Kotlin. Smooth 60fps UI, Material You support, adaptive layouts.

🌐

Self-Hosted

Your data, your servers. No cloud dependencies. Full control over your infrastructure.
